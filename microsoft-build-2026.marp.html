# 💻 GitHub Copilot App ## La experiencia de escritorio agente-nativa
El GitHub Copilot App es la nueva experiencia de escritorio construida sobre GitHub para el desarrollo agente-nativo. Desde una única vista "My Work", puedes ver todo el trabajo en movimiento.

🎯 My Work View

Desde una sola vista ves:

  • Sesiones activas
  • Issues conectados
  • Pull requests
  • Automatizaciones en segundo plano

🔧 Worktrees Aislados

Cada sesión corre en su propio git worktree:

  • Copia aislada de tu branch
  • Sin conflictos entre agentes
  • Gestión automática
  • Sin branch juggling
"El GitHub Copilot app ha proporcionado el centro de control que necesitábamos para el desarrollo agéntico."
— David Jobling, Avanade Inc.

# 🧠 Modelos MAI de Microsoft ## Los LLMs propios de Microsoft
Microsoft presentó 7 modelos de IA propios (MAI) en Build 2026, marcando un cambio estratégico hacia modelos in-house para competir directamente con Claude (Anthropic) y otros modelos de terceros.

📊 Los 7 Modelos MAI

Modelos in-house de Microsoft para diferentes cargas de trabajo:

  • Modelos de código
  • Modelos de reasoning
  • Modelos de grounding (Microsoft IQ)
  • Modelos multimodales

⚔️ Microsoft vs Anthropic

Los modelos MAI de Microsoft compiten directamente con Claude Code:

  • Mayor control sobre el stack
  • Integración nativa con Azure
  • Optimización para agentes

# 🤖 Agent 365 & Microsoft IQ ## Agentes inteligentes con grounding empresarial

🧠 Agent 365

  • Agentes que entienden el contexto empresarial

🌐 Microsoft Web IQ

Un nuevo estándar para el grounding de IA:
