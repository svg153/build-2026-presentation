🧬 Modelos MAI de Microsoft

Mustafa Suleyman (CEO de Microsoft AI) presenta 7 nuevos modelos de IA propios de Microsoft

  • Maia — LLM de lenguaje a escala GPT-5, optimizado para reasoning y coding
  • Mila — Modelo vision-language multimodal de última generación
  • Moataz — Especializado en reasoning complejo y planificación de agentes
  • MAI-Code-1 — Familia de modelos para código (Flash, Pro, Standard)
  • MAI-Code-1-Flash — El más rápido, ya integrado en Copilot
  • MAI-Research — Para análisis profundo y generación de documentación técnica
  • MAI-Small — Modelo eficiente para edge y dispositivos con recursos limitados

💡 Por qué importa: Microsoft ahora tiene su propia familia de modelos propietarios, reduciendo dependencia de OpenAI y creando un stack de IA vertical integrado.
