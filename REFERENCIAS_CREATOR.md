# 👤 Alex Graveley — Creador Original de GitHub Copilot

## 📖 Historia

### Quién es
- **Alex Graveley** — Creador de GitHub Copilot, Dropbox Paper, Hackpad, Tomboy Notes
- **Perfil:** Principal Engineer y Chief Architect detrás de GitHub Copilot
- **Web personal:** https://alexgraveley.com
- **Twitter:** @alexgraveley

### La controversia de compensación (Junio 2023)
- En junio de 2023, Alex Graveley publicó un tweet mostrando **su compensación total por crear GitHub Copilot: $20,000** (un bonus de $20K + un aumento de título)
- El tweet se hizo viral y generó **180+ comentarios en Hacker News**
- La respuesta de **Pete Hunt** (otro creador de GitHub): *"I made the same ($20k)"* — confirmando que la compensación fue similar

### Salida y retorno
- Cuando **anunció su salida**, Microsoft le ofreció **más que su promoción original + un bonus incentive** para quedarse
- **Se marchó de todas formas**
- La historia se discutió ampliamente en Reddit, Twitter y Hacker News
