# 🎨 Canvas en GitHub Copilot App

## 📖 Qué es Canvas
- **Canvas** es un panel interactivo integrado en GitHub Copilot App
- Permite ver y explorar el contexto de los agentes en tiempo real
- Los extensions pueden usar Canvas para mostrar información visual
- Soporta extensiones custom para augmentar el viewer

## 🌟 Ejemplo estrella: Quake 30 — Lee Reilly (@leereilly)
**Tweet:** https://x.com/leereilly/status/2068952875775283235

Lee Reilly abrió el source code de Quake en la Copilot App y creó un Canvas con:
- 🌐 Port to web + mobile
- 🤖 Copilot in the console (!)
- 🗺️ Procedurally generated levels
- 🎵 SoundCloud integration
- 📸 Screenshot + animated GIF capture
- 📺 CRT, ASCII, and other visual effects

**Demo:** https://leereilly.net/quakelike/
